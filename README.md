# JS_TrackBuilder
Web app for designing a race track / using html, CSS, JS

#Features 1.1

Size selection ranging from 20x20 grid to 100x100

Track Editor with different kinds of blocks
  - Grass
  - Road
  - Barricade
  - Finish
  - 
Option to save track into local storage in your browser
